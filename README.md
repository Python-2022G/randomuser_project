# RandomUser Project

practice for working with datetime 

## create an environment
```bash
python -m venv venv
```

## install requirements
```bash
pip install -r requirements.txt
```

## Create a RandomUser class

properties:

- url

methods:

- get_user_with_year(year)
- get_user_with_month(month)
